# Vendi Group Proposals

Static site hosting client proposals at `proposals.vendigroup.com`.

## Structure

- `/<client-slug>/index.html` — one folder per client proposal
- `CNAME` — custom domain for GitHub Pages
- `index.html` — minimal landing page (proposals reached by direct link only)

## Adding a new proposal

1. Create `/<client-slug>/index.html`
2. Commit and push to `master`
3. Reachable at `https://proposals.vendigroup.com/<client-slug>/`
